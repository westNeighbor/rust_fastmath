# Math solver lib for [Blender math anim addon]{https://github.com/westNeighbor/Blender_math_anim}

</div>

A fast Rust library for solving mathematical problems with Python API in Blender. This library provides functions 
for solving equations, performing calculus operations, and more. It's still under development, currently, it 
supports solving linear ODEs.

Use this to compile cross platform libs for Blender math anim addon.
